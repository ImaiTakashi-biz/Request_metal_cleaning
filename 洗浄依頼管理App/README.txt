# 洗浄依頼管理App - インストール・使用方法

## システム要件
- Windows 10/11 (64bit)
- メモリ: 最低 2GB RAM
- ディスク容量: 100MB以上の空き容量

## インストール方法

### 方法1: 単体実行（推奨）
1. `洗浄依頼管理App.exe` をお好みの場所にコピー
2. `config.json` を同じフォルダに配置
3. EXEファイルをダブルクリックして起動

### 方法2: デスクトップショートカット作成
1. `洗浄依頼管理App.exe` を右クリック
2. "ショートカットの作成" を選択
3. 作成されたショートカットをデスクトップに移動
4. ショートカットの名前を変更（オプション）

## 配布ファイル
- `洗浄依頼管理App.exe` - メインアプリケーション
- `config.json` - 設定ファイル（必須）
- `README.txt` - このファイル

## 注意事項
1. `config.json` は必ずEXEファイルと同じフォルダに配置してください
2. データベースファイルは初回起動時に自動作成されます
3. アンチウイルスソフトが警告する場合は、例外リストに追加してください

## アンインストール
- EXEファイルとその関連ファイル（データベースファイル等）を削除するだけです
- レジストリに情報は保存されません

## トラブルシューティング

### アプリが起動しない場合
1. `config.json` が同じフォルダにあるか確認
2. Visual C++ 再配布可能パッケージがインストールされているか確認
3. 管理者権限で実行してみる

### データが見つからない場合
1. データベースファイルの場所を確認
2. `config.json` のパス設定を確認
